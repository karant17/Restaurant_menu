Print-Ready Menu Builder

Editable restaurant menu with Excel import, live preview, QR code, and a print flow that saves cleanly as PDF.

Only Item and Price are required. Optional columns: Section, Description, Section Note, Image URL. If Section is missing, rows go into Menu Items.
The button opens a clean print view. In the print dialog, choose Save as PDF.

How to host on GitHub Pages

  1. Create a GitHub repository.
  2. Add one file named index.html.
  3. Paste this full code and commit to main.
  4. Go to Settings → Pages.
  5. Choose Deploy from a branch, then select main and /root.
  6. Open the GitHub Pages URL after it publishes.
  7. Build the menu, import Excel if needed, then click Download / Print PDF.

Use the live GitHub Pages URL in the QR field. Do not use a local file path. That is useless for customers.
