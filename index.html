Print-Ready Menu Builder

Built for a GitHub Pages workflow. Add sections manually or import them from Excel Sheet1, preview the layout, generate a QR code, then download a PDF for printing.

Use a sheet named Sheet1 or the first sheet. Only Item and Price are required. Optional columns: Section, Description, Section Note.
If your Excel has many items, import it instead of typing them one by one. The code will group rows by section automatically.
The button opens a clean print page. In the print dialog, choose Save as PDF or your printer.

How to host this on GitHub Pages

  1. Create a GitHub repository.
  2. Add a file named index.html.
  3. Paste this full code into it and commit.
  4. Go to Settings → Pages.
  5. Choose Deploy from a branch, then select main and /root.
  6. Wait for the public GitHub Pages URL.
  7. Open that URL and build the menu.
  8. Download the PDF and send it to the printer.

Use the GitHub Pages URL in the QR field. That is the link customers should scan.

The print flow is now independent of the live preview. If nothing appears, your browser is blocking popups. Allow popups for the site and try again.

Do not use a local file path. That is useless once the menu leaves your laptop.
